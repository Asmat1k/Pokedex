# Pokedex-app 
![Pokedex-app](https://github.com/user-attachments/assets/1968fc7d-d749-4bfa-9f1a-983e1ac7002c)

## Features
- User friendly errors format
- Loaders for pages and images
- Pagination
  -  Page pagination
  -  Endless scroll (with button)
- Error Boundary
- 404 Page
- Responsive layout
  
## Stack
- React
- React Router Dom
- TypeScript
- MobX
- SCSS
- Prettier|Eslint
- Webpack
- Custom architecture
